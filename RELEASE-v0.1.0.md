# Release v0.1.0

Prima release pubblica del progetto La Boutique della Bellezza.

## Highlights

- Setup iniziale del progetto Next.js con App Router e TypeScript.
- Homepage e catalogo prodotti pronti con navigazione per categorie.
- Hero aggiornata con immagine reale del salone come sfondo.
- Contesti applicativi per autenticazione e carrello inclusi.
- Componenti UI e pagine principali già versionati nel repository.
- Rimossi i riferimenti espliciti a v0 e Vercel dal progetto applicativo.

## Contenuto incluso

- Home page con hero, sezioni prodotto e footer.
- Pagine: admin, cart, checkout, login, ordini e dettaglio prodotto.
- Librerie dati e componenti condivisi per il catalogo.
- Asset pubblici del brand e immagine hero del salone.

## Note tecniche

- Build di produzione verificata con successo prima della pubblicazione.
- Repository Git inizializzato e branch `main` allineato con `origin/main`.
- Tag Git pubblicato: `v0.1.0`.
